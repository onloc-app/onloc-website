# Installation

The official Onloc app for Android currently has two installation methods. The app can either be installed from [F-Droid](https://f-droid.org/)
or by downloading the APK from the latest release on [GitHub](https://github.com/onloc-app/onloc-android/releases/latest/).

## F-Droid

Start by installing F-Droid via their [official installation](https://f-droid.org/en/docs/Get_F-Droid/).
Then, open the F-Droid app and search for Onloc. On Onloc's page, tap install.

</a>

## GitHub

Go to the [latest release](https://github.com/onloc-app/onloc-android/releases/latest/) of Onloc Android on GitHub and download the APK file (`onloc-{version}.apk`). Once the file is downloaded,
open it and follow the installation steps on your device.
